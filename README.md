# 基于Django的网络安全事件应急响应与处置系统

#### 介绍

一个利用后端Django+前端layui,数据库用Mysql的学生成绩管理系统。一个很不错的Django学习项目。

#### 功能

* 事件监控预警
* 安全事件通报
* 应急响应处置
* 数据业务恢复
